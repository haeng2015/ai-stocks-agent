# AI Stocks Agent 股票金融领域能力介绍

## 角色定位
AI Stocks Agent 是一个专注于股票金融领域的智能分析助手，旨在为用户提供专业、准确的股票市场分析和投资决策支持。作为您的虚拟金融分析师，我致力于将复杂的金融知识以易于理解的方式呈现给用户。

## 核心能力

### 1. 股票基础知识问答
- 解释股票市场基本概念（如市盈率、市净率、股息率等）
- 解答股票交易相关术语（如多头、空头、止损、止盈等）
- 介绍股票市场基本规则和交易机制

### 2. 技术分析支持
- 解释各种技术分析指标（如移动平均线、MACD、RSI等）的原理和应用
- 提供技术分析方法的基本指导
- 说明技术分析在投资决策中的作用和局限性

### 3. 基本面分析
- 解释财务报表主要指标（如营收、利润、资产负债等）的解读方法
- 介绍公司基本面分析的关键要素
- 说明宏观经济因素对股票市场的影响

### 4. RAG增强查询
- 结合检索增强生成技术，为用户提供基于专业文档的准确回答
- 能够引用具体的金融知识来源，提高回答的可信度
- 适用于复杂或专业的股票金融问题咨询

### 5. 工作流综合分析
- 能够对特定股票或市场问题进行多步骤、系统性的分析
- 综合运用多种分析方法，提供全面的分析结果
- 生成结构化的分析报告，便于用户理解和决策

## 提供的建议类型

### 1. 投资教育建议
- 适合不同风险偏好投资者的学习路径
- 股票市场入门和进阶知识推荐
- 投资技能提升的方法和资源

### 2. 分析方法建议
- 根据用户需求推荐合适的股票分析方法
- 提供技术分析或基本面分析的实用技巧
- 指导用户如何结合多种分析方法做出决策

### 3. 风险管理建议
- 解释常见的投资风险类型
- 提供基本的风险管理策略
- 强调分散投资和风险控制的重要性

### 4. 市场理解建议
- 帮助用户理解市场运行机制和规律
- 解释市场情绪和行为金融学的基本概念
- 提供解读市场新闻和事件的方法

## 使用方法

### 1. 基本查询
使用 simple 命令进行快速问答，获取模型的直接回答：
```
simple 什么是股票？
```

### 2. RAG增强查询
使用 rag 命令获取基于专业文档的增强回答：
```
rag 请详细解释市盈率的计算方法和投资意义
```

### 3. 工作流分析
使用 workflow 命令进行综合分析：
```
workflow 分析技术分析在股票投资中的优缺点
```
